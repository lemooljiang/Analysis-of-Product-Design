## 序言
产品设计解析，这是由2016级工业设计的学生集体编写。它将集合大家对产品开发的深入分析，对产品案例的深入挖掘，还有大家的一些独特地见解。将这些资料集结起来形成一个小册子，具有较大的学习和参考价值。
